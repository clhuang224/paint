# Paint

A minimalistic painting application built with HTML5 Canvas and JavaScript.

## Features

- Freehand drawing with adjustable brush size and color.
- Clear canvas functionality.
- Save artwork as an image file.
- Undo and redo actions.

## Tech Stack

- HTML5 Canvas
- JavaScript
- Alpine.js

## Usage

View [GitHub Pages](https://paint.lynns.me) to try the application.

## Development

### Install dependencies

```
bun install
```

### Run the development server

```
bun run dev
```

### Build for production

```
bun run build
```
